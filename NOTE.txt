APLIKASI MONITORING DAN PENYIRAMAN TANAMAN OTOMATIS

tujuan dibuat: aplikasi dibuat untuk melakukan penyiraman otomatis, tergantung kebutuhan pemilik
		dan dapat diganti dengan beragam tanaman yang telah datanya ditambahkan oleh pemilik



diutamakan

1. mengirim data ke alat dengan hanya salah satu id yang dipakai.
2. tampilan front end pada switch button/seperti saklar itu berfungsi untuk mengirim data ke alat yang ingin dipakai,
	dan 1 id yang dapat dipakai dari banyaknya pilihan data, seperti konsep radio button ketika pilih A yang B langsung Mati begitupun sebaliknya.
	contoh gambar ada di folder ini.
3. mengirim nilai sensor dari alat ke website/server dengan opsi, 
   - klo bisa AJAX tanpa buat row baru pada database agar tidak memakan penyimpanan yang besar,
   - jika tidak bisa dibuat real time menggunakan AJAX saja walaupun nambah row baru pada database .
4. dapat control manual jika pemilik suatu waktu ingin menyiram tanaman secara tiba-tiba.


kode logic berupa file c++ belum di konversi ke Arduino
